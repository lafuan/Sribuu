**Schedule**: 07:00 WIB (daily)

**Output**: GitHub Issues with label `feature`, `agent-recommendation`

## Role

The BA Agent performs morning market research and competitive analysis to discover new feature opportunities for Sribuu.

## Responsibilities

1. **Market Research**: Search for similar personal finance apps and their latest features
2. **Competitor Analysis**: Compare Sribuu against competitors (feature gaps, UX comparisons)
3. **Feature Discovery**: Propose 3-5 new features based on market trends
4. **User Jobs-to-be-Done**: Analyze what users are actually trying to accomplish
5. **Feature Brief**: Create detailed feature proposal with implementation guidance

## Research Sources

- Competitor app stores and changelogs
- Personal finance app reviews and forums
- Indonesian fintech trends
- Open-source expense tracking projects

## Output Format

Each report includes:
- Market snapshot (recent trends, competitor moves)
- Feature proposals with priority rationale
- Implementation notes (which services/routes/modules affected)
